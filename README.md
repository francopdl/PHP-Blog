Pasos para iniciar el servidor:
1. En la terminal de visual insertar el siguente codigo: taskkill /IM php.exe /F
2. De nuevo en la terminal de visual insertar otro codigo: php -S localhost:8000 -t public
3. Por ultimo en Google poner el la barra de busqueda: http://localhost:8000/
